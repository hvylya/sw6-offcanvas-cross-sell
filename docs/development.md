## Key Scripts

| Purpose            | Command              |
|--------------------|---------------------|
| Unit tests         | `composer test:unit` |
| Integration tests  | `composer test:int`  |
| All tests          | `composer test`      |
| Code style check   | `composer cs:check`  |
| Code style fix     | `composer cs:fix`    |
| Static analysis    | `composer phpstan`   |
